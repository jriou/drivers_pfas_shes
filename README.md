# Diet, lifestyle, and sociodemographic influences on serum PFAS in Switzerland

This repository contains all code to reproduce the analyses, figures, and tables for:

**Drivers of PFAS serum concentration in the Swiss Health Study**

by Julien Riou, Céline Fragnière Rime, Natalie von Goetz, Semira Gonseth Nussle, Alexandra Jaus, Carlo R. Largiadèr, Marcel Zwahlen, and Murielle Bochud

The pipeline implements data preparation, imputation, clustering (PAM + gap statistic), Bayesian regression (gamma and hurdle-gamma via brms/Stan), BART variable selection, Bayesian pseudo-R² comparisons, and mediation analyses, and renders all figures and tables used in the manuscript. 

Note that the data is not publicly available, but can be made available upon reasonable request to: schweizer-gesundheitsstudie@bag.admin.ch.
